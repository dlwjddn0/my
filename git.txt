1. git init
   - 새로운 Git 저장소(repository)를 생성한다.
   - 예시: git init

2. git add
   - 변경된 파일을 스테이지(stage)에 올린다.
   - 예시: git add hello.cpp

3. git commit
   - 스테이지에 올린 변경사항을 로컬 저장소에 기록한다.
   - 예시: git commit -m "첫 커밋 메시지"

4. git status
   - 현재 작업 디렉토리 상태를 확인한다 (추적된 파일, 변경된 파일 등).
   - 예시: git status

5. git push
   - 로컬 저장소의 변경 사항을 원격 저장소에 업로드한다.
   - 예시: git push origin main

6. git pull
   - 원격 저장소의 변경사항을 로컬로 가져온다.
   - 예시: git pull origin mai
