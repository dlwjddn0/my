# VS Code에서 자주 사용하는 확장 기능 목록 (3개 이상)

① Python (ms-python.python)  
→ Python 개발을 위한 기본 확장. Linting, IntelliSense, 실행 지원

② Code Runner (formulahendry.code-runner)  
→ 여러 언어의 코드 실행을 단축키로 빠르게 할 수 있게 도와줌

③ Bracket Pair Colorizer 2 (CoenraadS.bracket-pair-colorizer-2)  
→ 중괄호나 소괄호 쌍을 색으로 구분해 가독성 향상

④ Material Icon Theme (PKief.material-icon-theme)  
→ 프로젝트의 파일 아이콘을 직관적이고 예쁜 테마로 변경

⑤ Korean Language Pack for VS Code (MS-ceintl.vscode-language-pack-ko)  
→ VS Code를 한국어로 사용하고 싶을 때 유용한 확장

⑥ Prettier - Code formatter (esbenp.prettier-vscode)
→ 코드를 자동으로 예쁘게 정렬해주는 포맷터. 저장할 때 자동 정리도 가능

⑦ Live Server (ritwickdey.LiveServer)
→ HTML 파일을 실시간으로 웹 브라우저에 띄워서 즉시 확인 가능 (변경사항 자동 반영)

⑧ GitLens — Git supercharged (eamodio.gitlens)
→ Git 히스토리, blame, 변경 이력 등을 직관적으로 확인할 수 있음

⑨ Auto Rename Tag (formulahendry.auto-rename-tag)
→ HTML/XML 태그 이름을 수정하면 자동으로 짝 태그도 같이 수정됨

⑩ Path Intellisense (christian-kohler.path-intellisense)
→ 파일 경로 자동 완성 지원, import나 이미지 경로 작성 시 유용함
